Private Utility Locating

Know what’s below. Before every drill, core, or excavation — non-destructive subsurface mapping with paired RF locators.

Methodology

Paired locators. Cross-verified results.

Every JET mobilization begins with a private utility sweep using both Radiodetection RD 8000 and Metrotech 9860 XT — cross-verifying each target before marking.

We follow the APWA Uniform Color Code for temporary survey markings so any follow-on crew can read the site at a glance.

Before any intrusive work, Arizona 811 should also be contacted for public utility marking. JET integrates the 811 process into every project workflow.

Equipment

The instruments behind the sweep.

Radiodetection RD 8000

Multi-frequency active / passive locator — power, comms, and metallic pipe tracing.

Metrotech 9860 XT

Precision RF locator used alongside the RD 8000 for cross-verified results.

Metrotech 9890 RF Unit

Higher-power RF locator for deeper targets and more-congested subsurface environments.

APWA Uniform Color Code

Industry-standard marking colors applied on every JET mobilization.
